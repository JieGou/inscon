Performed research so that we may create custom item templates for future projects. 

The project consists of a ViewModel Item Template implementation and a VSIX host package that many more templates can be added to. The VSIX is an executable that will install the extension to the appropriate Visual Studio instance.

The following CodeProject article was used for reference: http://www.codeproject.com/Articles/365680/Extending-Visual-Studio-Part-3-Item-Templates
